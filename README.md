# 寂灭守护 · 余烬微光

> 一个 **AI Native** 的 2D 横版动作闯关网页游戏。
> NPC 由大语言模型（LLM）实时驱动，离开 AI 即无灵魂。

---

## 世界观

世界被遗忘的神明「寂主」引发枯寂侵蚀——不是邪恶毁灭，而是回收被人类截留的生机、修复生死循环。怪物是被侵蚀扭曲的生物残骸。世界 99% 生命已死亡，氛围压抑、枯萎褪色。

你是一名守护者，保护最后的「命种者」穿越废墟，前往寂灭神殿，重燃世界的余烬。

## 快速开始

### 环境要求

- 现代浏览器（Chrome / Edge），无需安装任何软件
- Node.js（仅用于启动本地服务器）

### 启动方式

```bash
# 1. 进入项目目录
cd d:/VScode/New_Project

# 2. 启动本地服务器（二选一）
node -e "const http=require('http'),fs=require('fs'),path=require('path');http.createServer((req,res)=>{const f=path.join('.',req.url==='/'?'index.html':req.url);try{const d=fs.readFileSync(f);res.writeHead(200,{'Content-Type':f.endsWith('.html')?'text/html':'application/octet-stream','Cache-Control':'no-cache'});res.end(d)}catch(e){res.writeHead(404);res.end('404')}}).listen(8080,()=>console.log('http://localhost:8080'))"

# 3. 浏览器打开
# http://localhost:8080/game.html
```

> 不要直接双击 HTML 文件！必须通过服务器访问，否则浏览器 CORS 策略会阻止 AI API 调用。

## 三关结构

| 关卡 | 名称 | 范围 | 怪物 | 特色 |
|------|------|------|------|------|
| 第一关 | 静谧海洋 | x:0~780 | 5枯骸 + 1精英 | 教学关，入门 |
| 第二关 | 暗黑森林 | x:780~1580 | 6怪 + 2波精英 | 多层平台跳跃 |
| 第三关 | 无尽虚空 | x:1580~2400 | 6守卫 + Boss寂灭守卫 | Boss三阶段 |

### 关卡机制

- **枯寂屏障**：进入关卡后两端被能量墙封锁，必须清完所有怪物才能前进
- **波次系统**：每关怪物分波激活，第1波全灭后第2波精英苏醒
- **存档复活**：清完一关自动存档，死亡从最近存档点复活（60% HP/MP）
- **关卡奖励**：清关后 HP+30%、MP+40%、掉落恢复球

## AI Native NPC 系统

### 核心理念

命种者（NPC）的每一个决定——**情绪、移动、对话、记忆**——全部由火山方舟大模型实时生成。代码中不预存任何对话文本，不做关键词匹配。API 断连时 NPC 真的沉默。

### 架构

```
每 3 秒：游戏状态 JSON → 火山方舟 API → LLM 返回 JSON 决策 → 驱动 NPC
```

LLM 返回的 JSON：
```json
{
  "emotion": "trust|fear|despair|dependence|worried|grateful",
  "thought": "内心独白",
  "reply": "对守护者说的话",
  "action": "follow|follow_close|stay|back_away",
  "targetX": 200,
  "memory": "值得记住的事"
}
```

### 信任值系统

信任值 0-100，影响 NPC 行为和 LLM 的情绪倾向：

| 事件 | 变化 |
|------|------|
| 玩家在 NPC 附近击杀怪物 | +3 |
| NPC 被怪物击中 | -5 |
| 玩家使用火种庇佑治疗 NPC | +8 |
| 玩家与 NPC 对话 | +2 |
| 清完一关 | +10 |
| NPC 坠落受伤 | -10 |
| 玩家远离 NPC 超过 250px | -1/秒 |

### API 配置

游戏内置火山方舟（Volcengine Ark）作为默认 API 提供商：

- API 地址：`https://ark.cn-beijing.volces.com/api/v3/chat/completions`
- 接入点 ID 和 API Key 已嵌入代码
- 点击右上角 🤖 可查看/修改配置
- 支持所有 OpenAI 兼容 API（DeepSeek / 通义千问 / Kimi 等）
- 状态指示灯：🟢 正常 | 🟡 等待 | 🔴 断连

## 玩家系统

### 属性

| 属性 | 初始值 | 说明 |
|------|--------|------|
| HP | 100 | 缓慢自动回复 |
| MP | 100 | 自动回复，击杀回蓝 +15 |
| 移速 | 1.7 px/帧 | 约 102 px/s |
| 跳跃高度 | ~195px | 可变高度（按住 K 更高） |

### 4 技能

| 键 | 技能 | MP | CD | 伤害 | 解锁 |
|----|------|-----|-----|------|------|
| Q | 疾风斩 | 15 | 0.6s | 32 | 进第2关 |
| U | 裂空斩 | 25 | 0.9s | 55 | 开局 |
| I | 寂灭波动 | 35 | 1.5s | 65 | 进第3关 |

### 战斗系统

- 三段连击（J 键）：连击窗口内连续按可打出递增伤害
- 击中停顿（Hitstop）：命中时画面冻结数帧，增强打击感
- 伤害飘字：白色数字浮起
- 屏幕震动：受伤/技能释放时画面抖动
- 技能刀光：裂空斩金色弧线、疾风斩蓝色冲刺线
- 护盾减伤：火种庇佑期间受伤减半

## 敌人系统

### 枯骸（普通怪）

巡逻于平台之上，玩家/NPC 踏上同一平台时激活追击。

### 侵蚀者（精英怪）

HP 更高、伤害更大、体型更大。击败后必定掉落 HP+MP 球。

### 寂灭守卫（Boss）

HP 350，三阶段：
- **P1**（HP>60%）：近战挥砍 + 冲刺
- **P2**（HP 25%-60%）：增加地面 AOE 裂波
- **P3**（HP<25%）：狂暴加速 + 召唤枯骸（最多 5 只）

## 技术栈

- **前端**：纯 HTML + CSS + JavaScript（Canvas 2D 渲染）
- **AI**：火山方舟 API（OpenAI 兼容）/ LLM 驱动 NPC
- **音效**：Web Audio API 振荡器合成 8-bit 音效
- **背景音乐**：HTML5 `<audio>` 标签
- **物理**：自研 2D 碰撞检测 + 重力 + 平台系统
- **渲染**：Canvas 2D，像素风角色，粒子特效，视差滚动背景
